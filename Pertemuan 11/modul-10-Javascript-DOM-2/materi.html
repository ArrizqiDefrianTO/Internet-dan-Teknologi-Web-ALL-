Modul 10 Javascript DOM 2

Event

Memungkinkan javascript untuk merespon interaksi dari user terhadap elemen yang ada di dalam web

Pada pertemuan ini kita akan bahas lebih lanjut penggunaannya, dan juga kita akan bahas event pada JavaScript yang akan kita gunakan untuk mempermudah pekerjaan kita

Event Handling
  • Fungsi yang dipanggil ketika sebuah event terjadi
  • Biasanya ketika ada interaksi dengan elemen HTML
  • Sebuah event harus didaftarkan

Mendaftarkan Event
  1. Inline

    • Inline event memerintahkan javascript untuk mengeksekusi sebuah fungsi, ketika sebuah event terjadi akibat dari interaksi user terhadap sebuah elemen HTML
    • Di bagian HTML

    • Di bagian JavaScript

  2. Fungsi Ekspresi

    • Event didaftarkan di dalam script javascript menggunakan fungsi ekspresi tanpa memodifikasi sintaks pada halaman HTML
    • Di bagian HTML
    • Di bagian JavaScript

  3. addEventListener

    • Event didaftarkan di dalam script javascript menggunakan method addEventListener yang ada di dalam elemen
    • Di bagian HTML
    • Di bagian JavaScript

Method String.Match()


Method String.match() diguakan untuk menemukan String menggunakan rumus regular expresion. Hasil akhir method ini adalah array yang berisi data seluruh string ditemukan.

document.getElementById(id)

  • digunakan untuk memilih / mengambil satu elemen HTML yang terdapat pada DOM
  • berbeda dengan document.getElementsByTagName(namatag), yang mengambil seluruh elemen dengan tag yang dimaksud dan menyimpannya sebagai array
  • jika terdapat lebih dari 1 elemen dengan nama id yang sama, maka hanya elemen pertama saja yang dianggap / terpengaruh, elemen lain diabaikan
  • kenapa harus kita ambil dan simpan ke dalam variabel? karena akan kita manipulasi setelahnya (diubah css-nya / ditambahkan event kepadanya)
element.addEventListener('event', function(){ // fungsi yang dilakukan // });
  • menambahkan event pada elemen 'tombol1' pada saat elemen tersebut di-'click'
  • event yang diberikan bisa bermacam-macam, berikut beberapa contohnya :
    1. click
    2. dblclick
    3. mouseover
    4. mouseenter
    5. mouseout
    6. blur
    7. focus
element.style.<namaStyle> = 'value';
  • menambahkan style background-color dan transition pada elemen 'kotak'
  • <namaStyle> diambil berdasarkan property pada CSS menggunakan jenis penulisan camelCase jika property lebih dari 1 kata
  • contoh: font-family menjadi fontFamily
